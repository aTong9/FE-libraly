1、栈（stack）后进先出
- 其限制是在仅允许在表的一端进行插入和删除运算，这一端被称为栈顶，另一端被称为栈底
  
  - 生活中举例：
  - 1、自助餐的托盘（最新放上去的，最先被客人拿走）
  - 2、收到很多邮件（实体的），从上往下依次处理这些邮件（最新到的邮件，最先处理）
  注意：不允许改变邮件的次序，比如从最小开始，或者最紧急的邮件，否则就不是栈结构了，而是队列或者优先队列结构
